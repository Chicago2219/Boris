## Описание проекта
Этот проект демонстрирует запуск простого Python-приложения внутри Docker-контейнера.  
Приложение состоит из файла `main.py`, который выводит сообщение **"Hello, world!"**.  
Зависимости перечислены в `requirements.txt`, а Dockerfile задает процесс сборки и запуск приложения.

## Файлы проекта
- **Dockerfile** — инструкции для сборки контейнера.
- **main.py** — основной Python-скрипт.
- **requirements.txt** — зависимости проекта.

## Сборка Docker-образа
Из корневой директории проекта выполните:

```bash
docker build -t my-python-app .

docker run --rm my-python-app
```

В терминале Вы увидете:
```bash
Hello, world!
```
